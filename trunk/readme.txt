WordPress payment gateway add-on to make the MANGOPAY-WooCommerce plugin compatible with Dokan.


== Description ==

MANGOPAY is a payment solution dedicated to web marketplaces available worldwide.

This is an add-on to the official WooCommerce payment gateway for MANGOPAY to make it compatible with the Dokan marketplace plugin.


== Installation ==

= Requirements =

* You will need to have the [WooCommerce](https://www.wordpress.org/plugins/woocommerce/) and [Dokan](https://wedevs.com/dokan/) plugins installed and activated

= Updating =

Automatic updates should work as normal; as always though, ensure you backup your site just in case.
